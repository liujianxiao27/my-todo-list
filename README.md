# 我的待办清单

一个简洁的任务管理工具，帮你记录和追踪日常待办事项。

## 功能

- 添加新任务
- 标记任务完成
- 删除任务
- 任务数据自动保存（刷新不丢失）
- 分类任务
- 搜索任务
- 统计任务
- 任务时效性

访问：[你的网址.zeabur.app](https://my-todo-app.zeabur.app/todo-v1-final.html)

- CSS
- JavaScript
- localStorage（本地存储）
